# popodatetime
#####popodatetime移动端日期时间选择控件

##[DEMO](http://po-po.github.io/popodatetime/)

##使用方法
```javascript
//日期时间选择
$(".datetime-1").popodatetime({
  container: ".box .screen"
});
//日期选择
$(".datetime-2").popodatetime({
  container: ".box .screen",
  time: false
});
//时间选择
$(".datetime-3").popodatetime({
  container: ".box .screen",
  date: false
});
```
